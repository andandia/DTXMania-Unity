# bjxa (Audio Decoder)

## 役割
`.xa` 形式の音声ファイルをデコードするためのライブラリです。リトルエンディアンの読み書き、XAフォーマットの解析、およびPCMデータへのデコード機能を提供します。

## 主要なクラス

### LittleEndian (内部静的クラス)
バイト配列からショート、整数などをリトルエンディアン形式で読み書きするためのユーティリティです。

### Format
XAファイルおよびRIFF(WAV)ファイルのヘッダー情報（チャンネル数、サンプルレート、ブロックサイズなど）を管理するクラスです。

### Decoder
XA形式の音声データを解析し、Unityで再生可能なPCMデータ（float配列）に変換するメインクラスです。

## Decoderクラスのメソッド一覧

### Decode(byte[] xa)
XAデータのバイト配列を受け取り、ヘッダーを解析して全データをデコードした `float[]` 配列を返します。

### ReadHeader(byte[] xa) (内部)
XAファイルのヘッダーを読み込み、マジックナンバーの確認、チャンネル数、サンプルレート、ビット数などの状態（`DecoderState`）をセットアップします。

### BlockInflater(uint bits) (静的・内部)
ビット数（4, 6, 8）に応じたデコード（インフレート）処理を行う関数を返します。各ビット数ごとのビットシフト演算によるPCM復元ロジックが含まれています。

### DecodeInflated(short[] pcm, int off, byte prof) (内部)
ADPCM形式の差分データを、以前のサンプル値とゲイン係数を用いて実際の振幅値に復元（予測符号化の復号）します。

### Decode(byte[] xa, int xaOff, float[] pcm, out long totalPcmBlocks)
バイト配列からブロック単位で順次デコードを行い、PCMデータを生成します。
- `short` 型のPCMデータを、Unityの `AudioClip` で使用される `-1.0 〜 1.0` の `float` 型に正規化して出力します。
