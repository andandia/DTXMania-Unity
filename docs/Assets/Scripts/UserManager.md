# UserManager

## 役割
ユーザー設定のリストを管理し、現在ログインしているユーザーの情報を保持するシングルトンクラスです。

## 主要なフィールド
- `Instance`: `UserManager`の唯一のインスタンスにアクセスするための静的プロパティ.
- `UserList`: 利用可能なユーザー設定（`UserSettings`）のリスト。
- `LoggedOnUser`: `UserList`の中で現在選択されている（ログインしている）ユーザー。

## メソッド一覧

### UserManager() (コンストラクタ)
デフォルトのユーザーとして「AutoPlayer」（全自動再生）と「Guest」（全手動プレイ）をリストに追加します。
Guestユーザーは、シンバル、ハイハット、スネア、バスドラム、各タムのオートプレイ設定がすべてfalseに設定されます。
