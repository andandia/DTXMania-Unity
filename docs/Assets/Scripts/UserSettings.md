# UserSettings

## 役割
個々のユーザーのゲーム設定を保持するクラスです。オートプレイ設定、スクロール速度、判定の許容範囲、レーンのカスタマイズ設定などが含まれます。

## 主要なフィールド
- `Id / Name`: ユーザーを識別するためのIDと名前。
- `ScrollSpeed`: ノーツが流れる速さの倍率。
- `AutoPlay_...`: 各ドラムパーツごとのオートプレイ（自動演奏）の有効/無効フラグ。
- `MaxRange_...`: PERFECT、GREAT、GOOD、OKの各判定の許容時間（秒）。
- `CymbalFree`: シンバルフリーモード（左右のシンバルを区別しないなど）の有効フラグ。
- `DrumChipProperty`: プレイモードや楽器の配置設定に基づいた、チップの描画特性を管理するオブジェクト。

## メソッド一覧

### AutoPlay(AutoPlayType type)
指定された楽器タイプに対してオートプレイが有効かどうかを返します。

### AutoPlayAllOn()
すべてのドラムパーツのオートプレイが有効（全自動状態）かどうかを判定します。

### MaxHitTime(JudgmentType type)
指定された判定タイプに対する最大許容時間（判定範囲）を返します。
