# Grade

## 役割
演奏の成績（スコア、コンボ、判定数、達成率、エキサイトゲージ、スキル値など）を計算・管理するクラスです。

## 主要なフィールド
- `Score`: 現在の合計スコア。
- `Combo`: 現在の連続ヒット数。
- `MaxCombo`: 1曲を通した最大コンボ数。
- `ExciteGauge`: ゲージの残量（0.0〜1.0）。
- `Achievement`: 達成率（％）。判定とコンボ数から計算されます。
- `Skill`: スキルポイント。達成率と曲の難易度から計算されます。
- `TotalNotes`: 判定対象となる総ノーツ数。
- `JudgeToHitCount`: 判定タイプごとのヒット数。

## メソッド一覧

### ApplyScoreAndSetting(Score score, UserSettings settings)
曲の開始時にスコアデータとユーザー設定を読み込み、総ノーツ数や曲の難易度を設定します。オートプレイ設定によって、どのノーツを判定対象に含めるかをフィルタリングします。

### AddHit(JudgmentType judge, int add = 1)
新しいヒット判定を追加し、スコア、コンボ、達成率、スキル値を再計算します。
- PERFECT/GREAT/GOODでコンボが増加し、OK/MISSでリセットされます。
- スコアはコンボ数に応じたボーナス（最大50コンボまで）が加算されます。

### AddJudgementType(JudgmentType judge)
エキサイトゲージの増減を処理します。判定が良いほどゲージが増え、MISSで大きく減少します。

### ReBuildIfDirty()
判定割合（各判定が全体の何％か）の整数値を計算します。端数処理を行っても合計が100％になるように調整ロジックが含まれています。
