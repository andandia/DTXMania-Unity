# KeyBindings

## 役割
キーボードのキーやMIDIデバイスのノート番号と、ドラムの入力タイプ（スネア、バスドラムなど）の対応付けを管理するクラスです。

## 主要なフィールド
- `KeyboardToDrum`: キーボードの入力（`IdKey`）を`DrumInputType`に変換する辞書。
- `MIDItoDrum`: MIDIの入力（`IdKey`）を`DrumInputType`に変換する辞書。

## 構造体: IdKey
デバイスIDとキー番号（またはノート番号）のペアを表します。

## メソッド一覧

### KeyBindings() (コンストラクタ)
デフォルトのキーアサインとMIDIアサインを初期設定します。
- **キーボード**: Q/Returnが左シンバル、Aがハイハットオープン、Xがスネア、Space/Cがバス、Mが右シンバルなど。
- **MIDI**: 一般的なMIDIドラムのノート番号（36: バス、38: スネア、42: ハイハットクローズなど）に基づいた割り当てが行われています。
