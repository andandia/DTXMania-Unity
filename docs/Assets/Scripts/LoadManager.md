# LoadManager

## 役割
ディレクトリの読み込みなどを管理するためのクラスですが、現時点では非常にシンプルな実装となっており、将来的な拡張のためのプレースホルダー的な役割を果たしています。

## 主要なフィールド
- `Instance`: `LoadManager`の唯一のインスタンスにアクセスするための静的プロパティ。

## メソッド一覧

### LoadDirectory(string dir, Action<string> onLoadDir)
指定されたディレクトリパスを受け取り、提供されたコールバック関数を即座に実行します。
現在は単にパスを渡すだけの処理を行っています。
