# SwitchManager

## 役割
画面遷移時のエフェクト（スイッチ）を管理するシングルトンクラスです。`StageManager`と似た構造を持ち、トランジション演出（シャッター、フェードなど）のライフサイクルを制御します。

## 主要なフィールド
- `Instance`: `SwitchManager`の唯一のインスタンスにアクセスするための静的プロパティ。
- `CurrentSwitch`: 現在実行中のスイッチ演出。

## メソッド一覧

### Open<T>()
指定した型`T`のスイッチ演出を開始します。
- 型名に対応するプレハブを`Resources`からロードしてインスタンス化します。
- スイッチオブジェクトを作成し、`OnOpen`を呼び出します。

### Update()
現在実行中のスイッチ演出の`Update`メソッドを呼び出します。

### Close(Switch stage)
演出が終了した際にスイッチを閉じ、GameObjectを破棄して`OnClose`を呼び出します。
