# UIAnimation

## 役割
UI要素のアニメーション（トゥイーン）を制御するための汎用クラスです。移動、回転、スケール、フェードなどの数値を、時間経過とともに特定のイージング関数に従って変化させる機能を提供します。

## 主要なフィールド
- `method`: イージングの種類（Linear, EaseIn, EaseOut, Bounceなど）。
- `style`: 再生スタイル（Once, Loop, PingPong）。
- `animationCurve`: カスタムのアニメーション曲線を使用する場合に設定します。
- `delay`: 再生開始までの遅延時間。
- `duration`: 再生にかかる時間。
- `onFinished`: アニメーション終了時に実行されるコールバック。
- `OnUpdate`: 値が更新された際に実行されるコールバック。実際の値の変化（透明度の変更など）をここで行います。
- `tweenFactor`: 現在の進行度（0.0〜1.0）。

## メソッド一覧

### Update()
毎フレーム呼び出され、進行度（`mFactor`）を更新し、それに基づいたイージング値を計算して`Sample`を呼び出します。ループやピンポン再生の制御もここで行います。

### Sample(float factor, bool isFinished)
進行度（0〜1）を受け取り、選択されたイージングメソッドに基づいて最終的な値を計算します。計算結果は`OnUpdate`イベントを通じて通知されます。

### PlayForward() / PlayReverse()
アニメーションを正方向または逆方向に再生開始します。

### Play(bool forward)
指定した方向にアニメーションを開始し、`mEnabled`をtrueにします。

### ResetToBeginning()
アニメーションの状態を開始地点（正方向再生なら0、逆方向なら1）にリセットします。

### Toggle()
現在の進行方向を反転させます。

### BounceLogic(float val)
バウンス（弾むような動き）のイージング計算を行う内部メソッドです。
