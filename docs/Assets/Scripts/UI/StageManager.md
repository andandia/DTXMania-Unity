# StageManager

## 役割
ゲームの各画面（ステージ）の遷移を管理するシングルトンクラスです。プレハブのロード、インスタンス化、および現在アクティブなステージのライフサイクル管理を行います。

## 主要なフィールド
- `Instance`: `StageManager`の唯一のインスタンスにアクセスするための静的プロパティ。
- `CurrentStage`: 現在表示されているステージ。

## メソッド一覧

### Open<T>()
指定した型`T`のステージを開きます。
- 型名と同じ名前のプレハブを`Resources`フォルダからロードします。
- ロードしたプレハブを`MainScript.Instance.UIRoot`の下にインスタンス化します。
- ステージオブジェクトを作成し、`OnOpen`を呼び出します。

### Update()
現在アクティブなステージの`Update`メソッドを毎フレーム呼び出します。

### Close(Stage stage)
指定したステージを閉じます。GameObjectの破棄と`OnClose`の呼び出しを行います。
