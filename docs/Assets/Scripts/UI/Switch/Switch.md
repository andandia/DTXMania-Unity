# Switch

## 役割
画面遷移時の演出（トランジション）を担当する基底クラスです。画面全体を覆うようなアニメーションを行い、演出の中間地点（画面が完全に隠れたタイミング）で実際のステージ切り替えを実行します。

## 主要なフィールド
- `mStageAnim`: 遷移演出用のアニメーションコンポーネント。
- `OnSwitchMiddleClosed`: 画面が完全に閉じられた（隠れた）タイミングで呼び出されるコールバック。
- `mCloseExecuted`: 切り替え処理が既に実行されたかどうかを管理するフラグ。

## メソッド一覧

### Switch() (コンストラクタ)
基底クラス `Activity` を初期化します。

### OnOpen()
演出を開始します。デフォルトでは、画面が開いた状態から閉じる方向のアニメーション（逆再生など）を `PlayCloseAnim` で開始します。

### Update()
アニメーションの終了を監視します。
1. 閉じるアニメーションが終わった際、`ExecuteSwitch` を呼び出してステージ切り替えを実行します。
2. その後、開くアニメーションが終わった際、このスイッチ自体を閉じます（破棄します）。

### PlayCloseAnim() / PlayOpenAnim() (仮想メソッド)
画面を閉じる（覆い隠す）アニメーション、および開く（中身を見せる）アニメーションを再生します。

### ExecuteSwitch() (仮想メソッド)
画面が完全に隠れた際に行う処理です。
- `OnSwitchMiddleClosed` コールバックを呼び出し、ステージの切り替えなどを行います。
- `PlayOpenAnim` を呼び出して画面を開く演出を開始します。

### Close()
`SwitchManager.Instance.Close(this)` を呼び出し、自身を破棄します。
