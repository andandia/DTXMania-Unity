# LaunchStage

## 役割
アプリ起動時に表示される初期化ステージです。楽曲ファイルの検索、プレビュー画像の読み込みなどを行い、完了後にタイトル画面へ遷移します。

## 主要なフィールド
- `mTextCount`: 読み込まれたファイル数や進捗を表示するためのUIテキスト。

## メソッド一覧

### OnOpen()
ステージが開かれた際に呼び出され、初期化プロセス（`DelayOpen`コルーチン）を開始します。

### GetExternalFilesDir() (静的メソッド)
Androidデバイスにおいて、外部ストレージ内の`DTXFiles`フォルダを検索します。Unityエディタ上では動作しません。

### DelayOpen() (IEnumerator)
楽曲の読み込み処理を順次実行します。
1. `MainScript`に設定されたフォルダおよび外部ストレージからDTXファイルを再帰的に検索します。検索処理は別スレッドで実行されます。
2. `StreamingAssets`フォルダ内の組み込みDTXファイルを読み込みます。
3. すべての楽曲ノードに対して、プレビュー画像（ジャケット画像など）を非同期で読み込みます。
4. すべての読み込みが完了したら、`TitleStage`を開き、自身を閉じます。
