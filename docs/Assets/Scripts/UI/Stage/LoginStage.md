# LoginStage

## 役割
ユーザー選択（ログイン）画面を制御するステージです。利用可能なユーザーリストからプレイするユーザーを選択します。

## 主要なフィールド
- `mUserList`: ユーザー選択用のトグルボタンが配置されているUIコンテナのTransform。
- `mPressedCancel`: キャンセル操作（終了操作）の状態管理用フラグ。

## メソッド一覧

### OnOpen()
ステージが開かれた際に、UI上のユーザーリストにイベントリスナーを登録し、現在選択されているユーザーのトグルをONにします。

### Update()
入力管理（`InputManager`）を通じてユーザーの操作を監視します。
- 上下移動: `UserManager.Instance.UserList`内の選択ユーザーを切り替えます。
- OK: `OnOK`を呼び出し、選曲画面へ遷移します。
- キャンセル: 2回連続でキャンセル操作が行われた場合、アプリケーションを終了します。

### ToogleSelectedUser()
`UserManager`で現在選択されているインデックスに基づいて、UI上のトグル表示を更新します。

### OnToggleValueChanged(bool selected)
UI上のトグルボタンがクリックされた際のコールバックです。選択されたユーザーを`UserManager`に反映させます。

### OnOK()
画面遷移演出（フェード）を開始し、演出の中間地点（画面が隠れたタイミング）で`SelectionStage`（選曲画面）を開き、自身のステージを閉じます。
