# ScoreDisplay

## 役割
演奏中の現在の合計スコアをリアルタイムに表示するクラスです。

## 主要なフィールド
- `mCurrentGrade`: スコア情報を取得するための `Grade` クラスへの参照。
- `mScoreAnim`: スコアが更新された際に再生される全体のアニメーション。
- `mNumberAnimInfo`: 各桁の数字を表示するテキストコンポーネントのリスト。

## メソッド一覧

### OnOpen()
UIの各桁のテキストコンポーネントを取得し、初期状態をセットアップします。

### Update()
毎フレーム呼び出され、`Grade` クラスのスコアに変更があった場合に表示を更新し、全体のアニメーションを再生します。

### UpdateScoreText(int displayValue)
数値を文字列に変換し、値が変化した桁のテキストを更新します。足りない桁は 'o'（グレーの0）で埋められます。
