# ChipSlot

## 役割
特定の種類のチップ（例: スネアチップのみ、小節線のみ）を描画するための「スロット（レーン）」を管理するクラスです。画面内の描画位置を更新し、チップの表示用オブジェクト（チップノード）の生成と再利用（オブジェクトプーリング）を行います。

## 主要なフィールド
- `mChipType`: このスロットが担当するチップの種類。
- `mChipTemplate`: チップ表示の雛形となるオブジェクト。
- `mChipNodeMap`: チップデータと表示用オブジェクト（Transform）の紐付け辞書。
- `mActiveChips`: 現在画面内に表示されているチップのキュー。

## メソッド一覧

### Update()
毎フレーム呼び出され、`PlayingStage` の描画リストに含まれるこのスロット担当の全チップの位置を更新します。
- 各チップの `PixelDistance` に基づいて、Y座標を設定します。
- 画面外に消えたチップを特定し、`CheckForOutTimeChip` によって非表示にします。

### CheckForOutTimeChip(Chip firstActiveChip)
描画リストに含まれなくなった（＝画面外に消えた、またはヒットされた）古いチップを非表示にし、管理リストから除外します。

### GetChipNode(Chip chip)
指定されたチップに対応する表示用オブジェクトを取得します。
- すでに紐付けられている場合はそれを返します。
- 新しく表示が必要な場合は、非アクティブなオブジェクトを再利用するか、テンプレートから新しく生成します。
- 小節線の場合は、小節番号をテキストに反映させます。
