# ChipLight

## 役割
ノーツ（チップ）をヒットした際に、そのトラック（レーン）の判定ライン付近を光らせる演出（ヒットライト）を管理するクラスです。

## 主要なフィールド
- `mTrackDrumMap`: トラックの種類（`DisplayTrackType`）と、対応する発光アニメーションの辞書。

## メソッド一覧

### OnOpen()
初期化時に子要素をすべて走査し、各オブジェクト名に対応する `DisplayTrackType` のアニメーションを辞書に登録します。初期状態ではすべて非表示にします。

### OnHit(DisplayTrackType displayTrackType)
指定されたトラックでヒットが発生した際に呼び出されます。対応するアニメーションオブジェクトをアクティブにし、最初から再生します。
