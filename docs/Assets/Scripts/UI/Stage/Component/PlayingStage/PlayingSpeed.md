# PlayingSpeed

## 役割
演奏中のスクロール速度（ハイスピード設定）を管理し、表示するクラスです。演奏中にリアルタイムで速度を変更する機能も備えています。

## 主要なフィールド
- `InterpSpeed`: 現在の描画に使用されている、補間されたスクロール速度。
- `mSpeedText`: 速度の数値を表示するテキストUI。
- `mSpeedSlider`: 速度を視覚的に表示するスライダーUI。

## メソッド一覧

### OnOpen()
UIコンポーネントの取得を行い、現在のユーザー設定（`ScrollSpeed`）を初期速度として設定します。

### Update()
毎フレーム呼び出され、以下の処理を行います。
1. `InterpSpeed` を実際のユーザー設定速度に近づけます（滑らかな速度変化のため）。
2. `CheckInput` により、演奏中の速度変更入力を監視します。

### CheckInput()
ユーザーが上または下のボタン（ドラム入力を含まない純粋なキー入力など）を押した場合に、スクロール速度を0.5刻みで増減させます。

### SetSpeed(float speed)
スクロール速度を新しい値に設定します。
- 最小0.5、最大8.0の範囲に制限します。
- ユーザー設定値を更新し、UI（テキストおよびスライダー）の表示を更新します。
