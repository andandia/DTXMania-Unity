# ComboDisplay

## 役割
演奏中の連続ヒット数（コンボ数）を表示し、コンボ数が増える際のアニメーションを制御するクラスです。

## 主要なフィールド
- `mCurrentGrade`: コンボ数の情報を取得するための `Grade` クラスへの参照。
- `mHundredAnim`: 10コンボごと、または100コンボごとの特別な演出用のアニメーション。
- `mNumberAnimInfo`: 各桁の数字（Text）と、その数字が変化した際の個別アニメーションを保持するリスト。

## メソッド一覧

### OnOpen()
UIの各桁のテキストとアニメーションコンポーネントを取得し、初期状態をセットアップします。

### Update()
毎フレーム呼び出され、`Grade` クラスのコンボ数に変更があった場合に `UpdateComboText` を呼び出して表示を更新します。

### UpdateComboText(int combo)
コンボ数の数値を文字列に変換し、UIに反映させます。
- **演出**: コンボが10増えるごと、または100増えるごとにルートのアニメーション（揺れや拡大など）を再生します。
- **個別更新**: 値が変化した桁のみ、個別の数字アニメーションを再生して数値を更新します。
