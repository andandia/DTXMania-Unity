# DrumPad

## 役割
演奏画面の下部に表示されているドラムのパッド（各レーンに対応するUI要素）を管理するクラスです。入力やヒットがあった際に、パッドが叩かれたようなアニメーション（沈み込みなど）を再生します。

## 主要なフィールド
- `mTrackDrumMap`: トラックの種類（`DisplayTrackType`）と、対応するパッドのアニメーションコンポーネントの辞書。

## メソッド一覧

### OnOpen()
UI上の各パッド要素をスキャンし、オブジェクト名に基づいて担当するトラックを判別して辞書に登録します。

### OnHit(DisplayTrackType displayTrackType)
指定されたトラックに対応するパッドのアニメーションを再生します。すでに再生中の場合は停止してから再生し直すことで、連続した叩き込みに対応します。
