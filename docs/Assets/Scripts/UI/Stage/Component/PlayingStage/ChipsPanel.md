# ChipsPanel

## 役割
演奏画面において、画面内を流れるノーツ（チップ）を表示するためのパネル全体を管理するクラスです。各楽器（チップ種別）ごとの描画スロット（`ChipSlot`）を生成・管理します。

## メソッド一覧

### OnOpen()
1. パネル内のすべての子要素を一旦非表示にします。
2. ユーザー設定の `DrumChipProperty` に基づき、現在のプレイモード（BASIC/EXPERT）で表示が必要なチップ種別を特定します。
3. 表示が必要な各チップ種別に対して、対応するUI要素を検索し、`ChipSlot` を子Activityとして追加します。
