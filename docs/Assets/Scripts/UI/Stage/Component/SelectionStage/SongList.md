# SongList

## 役割
選曲画面における楽曲リストのUI表示とリストのスクロール、楽曲の選択を管理するクラスです。楽曲ツリー（`MusicTree`）の状態に合わせて、UI上のリスト項目を更新し、選択時のアニメーションを行います。

## 主要なフィールド
- `mCursorPos`: リスト表示内でのカーソル（選択中アイテム）の位置（中央付近のインデックス）。
- `mMoveDown`: 直前の選択移動が下方向だったかどうかのフラグ（アニメーション方向に使用）。
- `mItemTemplate`: リスト項目のテンプレートとなるUI要素。
- `mUIAnimation`: リスト選択切り替え時のスクロールおよび選択強調アニメーションの制御。
- `TotalSongItem`: 画面上に同時に表示するアイテムの総数（10個）。

## メソッド一覧

### OnOpen()
UIの初期化を行い、`MusicTree`のフォーカス変更イベントを購読します。また、選択アニメーションの定義を行います。

### Update()
選択切り替えアニメーションを更新します。

### OnClose()
`MusicTree`のイベント購読を解除します。

### RefreshSongList()
`MusicTree`の現在のフォーカスに基づき、画面上に表示する全アイテムの情報を更新します。また、新しく選択された楽曲のプレビュー音声を再生します。

### BuildSongItem(Node node, int index)
個々の楽曲リスト項目（`SongItem`）の表示内容（タイトル、サブタイトル、サムネイル画像、座標）をセットアップします。

### SelectPrevious() / SelectNext()
前または次の楽曲を選択します。

### IntoBox()
現在選択中のアイテムが「BOX（フォルダ）」の場合、その中身の最初のアイテムにフォーカスを移動します。

### OutofBox()
現在のフォルダの親ノードにフォーカスを移動し、フォルダ外に出ます。
