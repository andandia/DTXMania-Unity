# SelectionStage

## 役割
選曲画面を制御する中心的なステージです。楽曲リストの表示、楽曲のプレビュー、難易度や成績の表示、および楽曲の選択・決定を管理します。

## 主要なフィールド
- `mSongList`: 楽曲リストのUIとロジックを管理するクラス。
- `mPreviewImage`: 選択中の楽曲のジャケット画像を表示するクラス。

## メソッド一覧

### OnOpen()
ステージ開始時に、楽曲リスト（`SongList`）、ジャケット画像表示（`PreviewImage`）、難易度・成績表示（`DifficultyAndGrades`）の各パーツを子Activityとして初期化します。

### Update()
毎フレーム呼び出され、`UpdateInput`によるユーザー操作の受付を行います。

### UpdateInput()
ユーザーからの入力を処理します。
- 上下移動: 楽曲リストの選択を前後に切り替えます。
- OK:
  - 選択中が「BOX（フォルダ）」なら、その中に入ります。
  - 選択中が「戻る」なら、親フォルダに戻ります。
  - 選択中が「楽曲」なら、その曲の演奏準備（`PlayingMusicNode`）を開始します。
- キャンセル:
  - 親フォルダがあれば戻り、なければログイン画面（`LoginStage`）へ戻ります。

### PlayingMusicNode(Node focusNode)
選択された楽曲の演奏を開始するための準備を行います。
- 現在再生中のプレビュー音声を停止します。
- 選択されたノードからスコアデータ（`Score`）を取得し、`MainScript`に保持させます。
- 「Ready Play Go」演出（`ReadyPlayGo`スイッチ）を開始し、その演出に合わせて曲読み込みステージ（`SongLoadStage`）へ遷移します。
