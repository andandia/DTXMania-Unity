# ResultStage

## 役割
演奏終了後の成績発表画面（リザルト画面）を制御するステージです。曲のタイトル、ジャケット画像、および判定数や達成率などの演奏成績を表示します。

## メソッド一覧

### OnOpen()
ステージが開かれた際に、演奏した曲の情報（タイトル、サブタイトル、ジャケット画像）をUIに反映させます。また、`PerformanceDispaly`を子Activityとして追加し、詳細な成績（PERFECT数など）を表示します。

### Update()
ユーザー入力を監視します。OK操作（決定）が行われると、シャッター演出（`ShutterSwitch`）を開始し、選曲画面（`SelectionStage`）へ戻ります。
