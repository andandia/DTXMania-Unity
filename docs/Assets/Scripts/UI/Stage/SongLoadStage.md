# SongLoadStage

## 役割
選曲後に、演奏に使用する音声ファイル（WAV）をメモリにロードするためのステージです。ロード完了後に演奏画面（`PlayingStage`）へ遷移します。

## メソッド一覧

### OnOpen()
ステージが開かれた際に、楽曲のジャケット画像を表示し、読み込みプロセス（`DelayOpen`コルーチン）を開始します。

### DelayOpen() (IEnumerator)
1. 画面遷移演出（スイッチ）が終了するのを待ちます。
2. 前の曲の音声データをクリアします。
3. 選択されたスコアの`WAVList`に基づき、すべてのWAVファイルを非同期で読み込み、`WAVManager`に登録（Sinup）します。
4. メモリを整理（UnloadUnusedAssets）し、少し待機してから演奏画面を開き、自身を閉じます。
