# SetNode

## 役割
`set.def` ファイルによって定義される、複数の難易度（ファイル）が一つの楽曲タイトルとしてまとめられたノードです。

## 主要なフィールド
- `MusicNodes`: 各難易度レベル（最大5つ）に対応する `MusicNode` の配列。

## メソッド一覧

### SetNode(SetDef.Block block, string baseFolder, Node parentNode) (コンストラクタ)
`set.def` の一つのブロック情報から、対応する各難易度の楽曲ファイルを読み込み、`MusicNodes` 配列を構築します。また、フォルダ内の特定の画像ファイルをサムネイルとして登録します。

### GetSelectMusicNode()
現在選択されている難易度、またはそれに最も近い利用可能な難易度の `MusicNode` を返します。

### PreviewSprite (プロパティ)
表示するプレビュー画像を返します。
- 現在フォーカスされている難易度の画像があればそれを優先します。
- なければ `SetNode` 自体の画像を返し、それでもなければ利用可能な難易度の画像を返します。
