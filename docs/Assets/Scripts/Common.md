# Common

## 役割
プロジェクト全体で使用される列挙型（Enum）や構造体を定義するファイルです。プレイモード、入力タイプ、判定タイプなどの基本的な定義が含まれています。

## 主要な定義

### PlayMode (Enum)
プレイの難易度設定などを表します。
- `BASIC`: 基本モード
- `EXPERT`: エキスパートモード

### DisplayTrackType (Enum)
演奏画面で表示されるトラック（レーン）の種類を定義します。
左シンバル、ハイハット、左ペダル、スネア、バス、各タム、右シンバルなどがあります。

### DisplayChipType (Enum)
演奏画面に流れてくるノーツ（チップ）の具体的な種類を定義します。
リムショットやゴーストノート、ライドのカップ、ミュートなど、詳細な打ち分けを区別します。

### DrumInputType (Enum)
ドラム入力デバイスからの入力の種類を定義します。
実際に判定に使用される入力タイプです。

### AutoPlayType (Enum)
オートプレイ設定の対象となる楽器の種類を定義します。

### InputGroupType (Enum)
入力のグループ化を定義します。シンバルフリーモードなどで使用されます。

### JudgmentType (Enum)
ノーツの判定結果を定義します。
- `PERFECT`, `GREAT`, `GOOD`, `OK`, `MISS`

## 主要な構造体

### LeftAndRightDisplayTrack
ライドシンバル、チャイナシンバル、スプラッシュシンバルを左右どちらのトラックに表示するかを保持する構造体です。
